# Niolesk

Niolesk is a Vite + React + TypeScript editor for Kroki diagrams.

## Development

```bash
npm install
npm run start
```

## Commands

```bash
npm run build
npm run preview
npm run test
npm run typecheck
```

## Runtime config

The default Kroki endpoint is configured in [public/config.js](/Users/darren/repos/niolesk/public/config.js).
